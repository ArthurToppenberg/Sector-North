# Radar image credits

The info window for a radar shows a photograph of the real Danish site, where one is
available under a licence we can use. Images are downscaled for the bundle (originals
linked below) but kept in colour. Attribution is also surfaced in-game as a small caption
on each image, as required by the CC BY-SA licences.

| Radar site | Image | Author | Licence |
| --- | --- | --- | --- |
| Skagen | [Radarsite Skagen 2.jpg](https://commons.wikimedia.org/wiki/File:Radarsite_Skagen_2.jpg) | heb | CC BY-SA 3.0 |
| Bornholm | [RytterknaegtenRadar.jpg](https://commons.wikimedia.org/wiki/File:RytterknaegtenRadar.jpg) | Elgaard | CC BY-SA 4.0 |

Both are photographs of the actual installations (Radarhoved Skagen; the radar at
Rytterknægten, Almindingen).

## ShareAlike note

Both licences are **CC BY-SA**: the downscaled copies bundled here are derivatives and
remain under the same licence. Using them (including in a paid build) is fine with the
attribution above; the images themselves stay CC BY-SA.

## Sites without an image

**Skrydstrup**, **Aalborg**, and **Karup** have no photo we have permission to use, so
their windows show the "NO IMAGE" placeholder. Add one by dropping the file in this
folder and registering it in `src/game/radarImages.ts` (texture key + credit). A real
Skrydstrup photo exists (Air Control Wing) but needs permission before use.
