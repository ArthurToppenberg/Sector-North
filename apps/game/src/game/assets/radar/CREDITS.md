# Radar image credits

The info window for a radar shows a photograph of the real Danish site, where one is
available under a licence we can use. Images are downscaled for the bundle (originals
linked below) but kept in colour. Attribution is also surfaced in-game as a small caption
on each image, as required by the CC BY-SA licences.

| Radar site | Image | Author | Licence |
| --- | --- | --- | --- |
| Skagen | [Radarsite Skagen 2.jpg](https://commons.wikimedia.org/wiki/File:Radarsite_Skagen_2.jpg) | heb | CC BY-SA 3.0 |
| Bornholm | [RytterknaegtenRadar.jpg](https://commons.wikimedia.org/wiki/File:RytterknaegtenRadar.jpg) | Elgaard | CC BY-SA 4.0 |
| Skrydstrup Air Base | [TPS 77 Romania.png](https://commons.wikimedia.org/wiki/File:TPS_77_Romania.png) | Revista Cer Senin | CC BY-SA 3.0 |

Skagen and Bornholm are photographs of the actual installations (Radarhoved Skagen; the
radar at Rytterknægten, Almindingen). **The Skrydstrup image is not the Danish
installation**: no freely licensed photo of it exists (a real one is held by the Air
Control Wing but needs permission), so the window shows a Romanian Air Force unit of the
same Lockheed Martin TPS-77 model — the sensor, not the site.

## ShareAlike note

All three licences are **CC BY-SA**: the downscaled copies bundled here are derivatives
and remain under the same licence. Using them (including in a paid build) is fine with
the attribution above; the images themselves stay CC BY-SA.

## Sites without an image

**Aalborg** and **Karup** have no photo we have permission to use — Commons has neither
the Danish installations nor any photo of their Indra PSR 2D model (checked 2026-07-22)
— so their windows show the "NO IMAGE" placeholder. Add one by dropping the file in this
folder and registering it in `src/game/radarImages.ts` (texture key + credit).
