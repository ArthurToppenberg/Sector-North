# Airfield image credits

The info window for an airfield shows a photograph of the field (terminal, tower,
runway or aircraft on it), where one is available under a licence we can use. Images are
downscaled for the bundle (originals linked below) but kept in colour. Where the licence
requires attribution it is also surfaced in-game as a small caption on each image (the
CC BY / CC BY-SA licences); the CC0 / public-domain images carry no obligation but are
credited here anyway.

## ShareAlike note

Most images are **CC BY-SA**: the downscaled copies bundled here are derivatives and
remain under the same licence. Using them (including in a paid build) is fine with the
attribution above; the images themselves stay CC BY-SA. The CC BY images require only
the attribution; the Sønderborg (CC0) and Lolland Falster (public domain) images carry
no obligation, but are credited above and in `src/game/airportImages.ts` as a courtesy.

## Fields without an image

The remaining airfields — mostly minor grass strips, glider fields and flying clubs —
have no photo on Wikimedia Commons under a licence we can use (checked 2026-07-22;
notably Kolding Lufthavn, Skive Lufthavn and Viborg Flyveplads only have GFDL-1.2-only
photos, which we reject). Their windows show the "NO IMAGE" placeholder. Add one by
dropping the file in this folder and registering it in `src/game/airportImages.ts`
(texture key + credit), keyed by the airport's exact `name` in `src/data/airports.json`.
